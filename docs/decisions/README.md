# Decision Records

Store durable project decisions here when they do not belong in an approved version, feature, or step design document.

- Use one Markdown file per decision.
- Name files `NNNN-short-title.md` with a stable, increasing number.
- Record context, decision, consequences, status, and links to the governing reviewed design.
- Do not use this directory for current state, execution logs, or temporary plans.
